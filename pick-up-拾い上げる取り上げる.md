# pick up (拾い上げる・取り上げる)

## 基本情報
**品詞**: 物目的用法単一目的語分離句動詞
**意味**: 拾い上げる・取り上げる（物目的用法）
**TOEIC重要度**: 90/100

## 文法的特徴
- **構文**: SVO（他動詞用法）
- **語順**: 分離可能（pick up the book / pick the book up）
- **目的語**: 物・書類・荷物（documents, packages等）

## 例文
"Please pick up the documents from the reception desk."
「レセプションデスクから書類を取ってください。」
