# run out from (飛び出す・突き出る)

## 基本情報
**品詞**: 単一用法三語句動詞
**意味**: 飛び出す・突き出る（単一用法）
**TOEIC重要度**: 30/100

## 文法的特徴
- **構文**: SV（自動詞用法）
- **語順**: 分離不可（前置詞句動詞）
- **目的語**: fromの後に飛び出し元（ventilation system等）
