# back up (バックアップする)

## 基本情報
- **表現**: back up
- **意味**: バックアップする・支援する
- **分類**: 句動詞 (前置詞句動詞)
- **TOEIC優先度**: 100
- **Ankiラベル**: 行動操作系前置詞句動詞

## 例文
Please back up all important files before the update.
更新前に重要なファイルをすべてバックアップしてください。

Part 4: IT関連業務指示
