# throw away (物を捨てる)

## 基本情報
**品詞**: 物目的用法単一目的語分離句動詞
**意味**: 物を捨てる・処分する
**TOEIC重要度**: 80/100

## 文法的特徴
- **構文**: SVO（単一目的語分離句動詞）
- **語順**: 分離可能（throw + 目的語 + away / throw away + 目的語）
- **目的語**: 物理的な物体・書類・ゴミ等

## 例文
"Please throw away these old documents in the shredder."
「これらの古い書類をシュレッダーで処分してください。」
