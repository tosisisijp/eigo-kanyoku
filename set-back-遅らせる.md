# set back (遅らせる)

## 基本情報
- **表現**: set back
- **意味**: 遅らせる・妨げる・後退させる
- **分類**: 分離可能句動詞
- **TOEIC優先度**: 85
- **Ankiラベル**: 分離句動詞副詞

## 例文
The rain set back our construction schedule by two weeks.
雨で建設スケジュールが2週間遅れました。
