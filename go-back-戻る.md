# go back (戻る・帰る)

## 基本情報
**品詞**: 自的句動詞
**意味**: 戻る・帰る（単一用法）
**TOEIC重要度**: 85/100

## 文法的特徴
- **構文**: SV（自動詞用法）
- **分離**: 不可能（固定句動詞）
- **目的語**: 通常は場所を表す前置詞句

## 例文
"I need to go back to the office to pick up some files."
「ファイルを取りにオフィスに戻る必要があります。」
