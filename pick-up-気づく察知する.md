# pick up (気づく・察知する)

## 基本情報
**品詞**: 節目的用法単一目的語分離句動詞
**意味**: 気づく・察知する（節目的用法）
**TOEIC重要度**: 75/100

## 文法的特徴
- **構文**: SV+that節（他動詞用法）
- **語順**: 固定（pick up that...）
- **目的語**: 節・状況・変化（that節、情報等）

## 例文
"The manager picked up that the project was behind schedule."
「マネージャーはプロジェクトが遅れていることを察知しました。」
